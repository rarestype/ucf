## References

This repository contains the specifications and reference implementation for the [Swift Unified Codelink Format](https://swiftinit.org/docs/ucf/help/codelinks) (UCF).

- [Symbol Link Help](https://swiftinit.org/docs/ucf/help/codelinks)
- [Syntax Grammar](https://swiftinit.org/docs/ucf/ucf/grammar)


## Requirements

The ucf library requires Swift 6.0 or later.
